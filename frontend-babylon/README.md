# Babylon.js Project

A Babylon.js project setup with TypeScript, Vite, and Tailwind CSS.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:3001`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run serve
```

## Project Structure

- `src/` - Source code
  - `main.ts` - Entry point
  - `App.ts` - Main application class
  - `style.css` - Global styles
- `public/` - Static assets
- `index.html` - HTML entry point

## Technologies

- Babylon.js 8.33.2
- TypeScript
- Vite
- Tailwind CSS
- Havok Physics
